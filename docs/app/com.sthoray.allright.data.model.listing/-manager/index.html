Manager

Model for a listing manager.

Contains information about the seller of a particular listing.

A manager might be an actual person such as a seller on the "secondhand" marketplace or a business store in the "mall" market place. This means the properties must be accessed with care depending on the marketplace.

Many fields are still missing from this data class and will likely need to be added. The "active_offer" field could be particularly useful for the listing activity.
