class SearchItem

Search item model.

This model (currently) does not contain all possible properties. As a general rule, we should process as little information as possible. Properties may not be used in both "Mall" and "Second hand" marketplaces. Differences also exist between categories. This is why some properties are nullable. TODO: Document search data properties
